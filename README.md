## 1. 프로젝트 개요  
### 1.1 프로젝트 소개

<h3>‘13ook’ 서비스의 이름은 숫자 ‘13’과 단어 ‘book’을 합쳐 만들어진 명칭 입니다.</h3>
<p>자신이 읽은 책에 대해 종이와 펜이 없더라도 언제 어디서든 편히 생각을 기록할 수 있으며,

도서별 짧은 리뷰와 책 속 명언을 적을 수 있는 하이라이트 페이지를 통해 다양하게 활용할 수 있는 서비스 입니다.</p>
